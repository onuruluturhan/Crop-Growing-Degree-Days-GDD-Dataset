# FAO-based crop temperature thresholds (T_base & T_upper) for Growing Degree Days (GDD) calculations.
## 🌡 GDD Formula

GDD = ((Tmax + Tmin) / 2) - T_base

## 🌿 Classification Note

This dataset follows agricultural classification (FAO):

- Okra → Vegetable crop  
- Bean → Grain legume  

Classification is based on usage, not strict botany.

## 📚 Source

FAO (2025) – Crop evapotranspiration  
Table 6.10 (p. 204)
